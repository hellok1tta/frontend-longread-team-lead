# История изменений (Changelog)

Все значимые изменения в проекте "CoffeeHouse" будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 10.02.2026

### 🎉 Первый релиз

#### Добавлено (Added)
- **Навигация**
  - Фиксированная навигационная панель с эффектом при скролле
  - Плавная прокрутка к секциям
  - Адаптивное меню для мобильных устройств

- **Главный экран (Hero)**
  - Полноэкранный баннер с фоновым изображением
  - Анимированный текст и кнопка призыва к действию
  - Кнопка для перехода к меню

- **Меню кофейни**
  - Категории: "Кофе" и "Десерты"
  - Карточки товаров с изображениями
  - Названия, описания и цены
  - Эффекты при наведении
  - Адаптивная сетка (grid layout)
- **О нас**
  - История кофейни с 2015 года
  - Преимущества в виде иконок
  - Фотография интерьера

- **Контакты**
  - Контактная информация (адрес, телефон, email, часы работы)
  - Форма обратной связи с валидацией
  - Ссылки на социальные сети

- **Подвал (Footer)**
  - Краткая информация о кофейне
  - Часы работы
  - Контактные данные
  - Копирайт

- **Стилизация**
  - Современная цветовая схема (карамельный, коричневый, кремовый)
  - Использование шрифтов: Playfair Display (заголовки), Inter (текст)
  - Адаптивный дизайн для всех устройств
  - CSS анимации при загрузке и наведении

- **Технические особенности**
  - Сборка на Vite для быстрой разработки
  - Компонентная архитектура React
  - Разделение стилей внутри компонентов
  - Оптимизация изображений через Unsplash CDN
